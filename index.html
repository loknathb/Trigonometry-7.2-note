টপিক ২: নির্দিষ্ট কোণের মান নির্ণয়

উদাহরণ: $tan 15^{\circ}$ এর মান নির্ণয়:

$tan 15^{\circ} = tan(45^{\circ} - 30^{\circ})$
$= \frac{tan 45^{\circ} - tan 30^{\circ}}{1 + tan 45^{\circ} \cdot tan 30^{\circ}} = \frac{1 - \frac{1}{\sqrt{3}}}{1 + 1 \cdot \frac{1}{\sqrt{3}}} = \frac{\frac{\sqrt{3}-1}{\sqrt{3}}}{\frac{\sqrt{3}+1}{\sqrt{3}}}$
$= \frac{\sqrt{3}-1}{\sqrt{3}+1} = \frac{(\sqrt{3}-1)(\sqrt{3}-1)}{(\sqrt{3}+1)(\sqrt{3}-1)} = \frac{3 - 2\sqrt{3} + 1}{3-1} = \frac{4-2\sqrt{3}}{2} = 2 - \sqrt{3}$

টপিক ৩: যৌগিক কোণের মান নির্ণয়

উদাহরণ: যদি $sin A = \frac{3}{5}$ এবং $cos B = \frac{12}{13}$ হয় (A, B সূক্ষ্মকোণ), $cos(A+B)$ এর মান নির্ণয় কর।
সমাধান:
$sin A = \frac{3}{5} \implies cos A = \sqrt{1 - sin^2 A} = \sqrt{1 - (\frac{3}{5})^2} = \frac{4}{5}$
$cos B = \frac{12}{13} \implies sin B = \sqrt{1 - cos^2 B} = \sqrt{1 - (\frac{12}{13})^2} = \frac{5}{13}$
$cos(A+B) = cos A cos B - sin A sin B = \frac{4}{5} \times \frac{12}{13} - \frac{3}{5} \times \frac{5}{13} = \frac{48-15}{65} = \frac{33}{65}$

টপিক ৫: $tan$ সম্পর্কিত অভেদ প্রমাণ

উদাহরণ: প্রমাণ কর যে, $tan 20^{\circ} + tan 25^{\circ} + tan 20^{\circ} tan 25^{\circ} = 1$
সমাধান:
আমরা জানি, $tan 45^{\circ} = 1$
বা, $tan(20^{\circ} + 25^{\circ}) = 1$
বা, $\frac{tan 20^{\circ} + tan 25^{\circ}}{1 - tan 20^{\circ} tan 25^{\circ}} = 1$
বা, $tan 20^{\circ} + tan 25^{\circ} = 1 - tan 20^{\circ} tan 25^{\circ}$
$\therefore tan 20^{\circ} + tan 25^{\circ} + tan 20^{\circ} tan 25^{\circ} = 1$ (প্রমাণিত)

টপিক ৬: ত্রিভুজের কোণ সম্পর্কিত সমস্যা ($A+B+C = \pi$)

উদাহরণ: যদি $A+B+C=\pi$ এবং $cos A = cos B cos C$ হয়, তবে প্রমাণ কর যে, $cot B cot C = \frac{1}{2}$
সমাধান:
$cos A = cos B cos C$
বা, $cos(\pi - (B+C)) = cos B cos C$
বা, $-cos(B+C) = cos B cos C$
বা, $-(cos B cos C - sin B sin C) = cos B cos C$
বা, $-cos B cos C + sin B sin C = cos B cos C$
বা, $sin B sin C = 2 cos B cos C$
বা, $\frac{sin B sin C}{cos B cos C} = 2 \implies tan B tan C = 2$
$\therefore cot B cot C = \frac{1}{2}$ (প্রমাণিত)

টপিক ৭: শর্তমূলক অভেদ

উদাহরণ: যদি $tan \alpha - tan \beta = x$ এবং $cot \beta - cot \alpha = y$ হয়, প্রমাণ কর যে, $cot(\alpha - \beta) = \frac{1}{x} + \frac{1}{y}$
সমাধান:
$y = cot \beta - cot \alpha = \frac{1}{tan \beta} - \frac{1}{tan \alpha} = \frac{tan \alpha - tan \beta}{tan \alpha tan \beta} = \frac{x}{tan \alpha tan \beta}$
$\implies tan \alpha tan \beta = \frac{x}{y}$
এখন, $cot(\alpha - \beta) = \frac{1}{tan(\alpha - \beta)} = \frac{1 + tan \alpha tan \beta}{tan \alpha - tan \beta} = \frac{1 + \frac{x}{y}}{x} = \frac{\frac{y+x}{y}}{x} = \frac{x+y}{xy} = \frac{1}{y} + \frac{1}{x}$ (প্রমাণিত)
